# DECODELABS-Task-1

## 📌 Project Overview
This project is part of my UI/UX Design internship at DecodeLabs. The goal of this task is to understand the target users before starting the design process. I created a User Persona and Empathy Map for a fitness application designed for busy office professionals.

## 🎯 Objective
- Identify the target audience.
- Understand users' goals, needs, and frustrations.
- Create a User Persona.
- Build an Empathy Map.
- Define the final UX research outcome.

## 🛠 Tools Used
- Figma

## 📖 Task Description
A fitness startup wants to launch an app for busy office professionals who struggle to find time for exercise. Before designing the app, UX research was conducted to understand users' pain points, motivations, and behaviors. This research helps in creating a user-centered and effective fitness solution.

# 🛒DECODELABS TASK 2
Low-Fidelity E-Commerce Homepage Wireframe

## 📖 Project Overview

This project is a Low-Fidelity Wireframe for an E-Commerce Homepage created in Figma. The wireframe focuses on layout structure, user flow, content hierarchy, and navigation before applying visual design.

---

## 🎯 Project Objective

The objective of this project is to design a simple and user-friendly homepage that helps users easily browse products, explore categories, and complete their shopping journey.

---

## 🛠️ Design Tool

- Figma

---

## 📱 Wireframe Features

- Announcement Bar
- Navigation Bar
- Hero Section
- Call-to-Action Button
- Featured Products
- Shop by Categories
- Promotional Offer Banner
- Customer Testimonials
- Newsletter Subscription
- Footer with Quick Links

---

## 🧩 User Flow

1. User lands on the homepage.
2. Browses featured products.
3. Explores product categories.
4. Views promotional offers.
5. Reads customer reviews.
6. Subscribes to the newsletter.
7. Continues shopping.

---

## 🎨 Design Principles

- Low-Fidelity Wireframe
- Simple Layout
- Clear Information Hierarchy
- Easy Navigation
- User-Centered Design
- Responsive Structure
