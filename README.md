# DECODELABS-Task-1

## 📌 Project Overview
This project is part of my UI/UX Design internship at DecodeLabs. The goal of this task is to understand the target users before starting the design process. I created a User Persona and Empathy Map for a fitness application designed for busy office professionals.

## 🎯 Objective
- Identify the target audience.
- Understand users' goals, needs, and frustrations.
- Create a User Persona.
- Build an Empathy Map.
- Define the final UX research outcome.

## 🛠 Tools Used
- Figma

## 📖 Task Description
A fitness startup wants to launch an app for busy office professionals who struggle to find time for exercise. Before designing the app, UX research was conducted to understand users' pain points, motivations, and behaviors. This research helps in creating a user-centered and effective fitness solution.
